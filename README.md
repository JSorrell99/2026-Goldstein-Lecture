# 2026 Jon Goldstein Memorial Lecture — Performance Report

A data visualization report for the 2026 Jon Goldstein Memorial Lecture hosted by the Department of Economics at the University of Minnesota.

## About

This report summarizes key performance metrics for the 2026 lecture featuring Richard Sandor, including email campaign open rates, event attendance, and web traffic analytics.

## What's Included

- Email open rate trend from 2020–2026
- Send-by-send breakdown of the 2026 campaign
- In-person and Zoom attendance data
- Event page web traffic and scroll depth analytics
- Historical open rate comparison table
